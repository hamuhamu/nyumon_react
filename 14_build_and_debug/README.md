# 14章 ビルド & デバッグ

## Browserify

Browserifyの実行時に--debug をつけることで、ソースマップ付きのファイルを出力する。
ソースマップは、連結前と連結後のファイルの行情報を保持するので、デバッグが可能となる。

## Watchify

watchifyを使うことで、ファイルの更新時に自動的にビルドを実行できる。
